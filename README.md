# Sales Performance Dashboard

**Tools Used:** Python (Pandas, Matplotlib, Seaborn), Power BI  
**Dataset:** Sample sales transactions for 2023  

## Overview
Analyzed regional and category-wise sales performance using Python and visualized KPIs in Power BI.  
The project identifies top-performing regions, profit trends, and sales distribution across categories.

## Key Insights
- North region showed highest overall sales.
- Electronics category contributed maximum profit.
- Seasonal peaks observed in Q3 and Q4.

## Skills Demonstrated
- Data cleaning and preprocessing with Pandas  
- Visualization using Matplotlib & Seaborn  
- Business insights communication using dashboards
